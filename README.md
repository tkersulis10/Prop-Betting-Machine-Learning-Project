NBA Prop Bet Analyzer

Provides three models for the user to use to predict NBA prop betting lines for points, rebounds, and assists for the 2023 NBA season.

To run the interactive user interface, use:

Windows:
python GUI.py

Mac/Linux:
python3 GUI.py
